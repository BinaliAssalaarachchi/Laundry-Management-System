# Payment Module & Database – Laundry Management System
Year 2 Semester 1 – Group Project

## Overview
This project implements the payment module and database integration for a Laundry Management System.  
It includes full Java backend code, frontend interfaces, database connectivity, and detailed reports.  

## Implementation & Tools and Technologies

- **Programming Languages:**  
  - Backend: Java 17  
  - Frontend: HTML, CSS, JavaScript, Thymeleaf  

- **Frameworks & Libraries:**  
  - Spring Boot  
  - Maven  
  - Thymeleaf  
  - Spring Data JPA  
  - Spring Security  
  - Spring Web  

- **Server:**  
  - Apache Tomcat (embedded with Spring Boot)  

- **Database:**  
  - MySQL  

- **IDE:**  
  - IntelliJ IDEA Ultimate Edition  

- **Deployment:**  
  - Deployed on a local server for development and testing  

- **Version Control:**  
  - GitHub  

## Notes
- This project demonstrates database integration, payment processing, and frontend-backend connectivity in a Laundry Management System.  
- All code is version controlled using GitHub.
